Hardening Active Directory Graphs

Active Directory (AD) systems face persistent cyber threats, yet traditional defenses lack adaptability. This project addresses edge-blocking inefficiencies using a Stackelberg game model, where an RL-driven attacker and an RL-assisted Evolutionary Diversity Optimization defender refine strategies through adaptive gameplay. To enhance efficiency, an RL Training Facilitator prunes environments and neural networks, improving scalability. Results demonstrate stronger AD security and enhanced scalability, making this a robust defense framework.

Structural Hole Spanners: Key Connectors in Networks

Structural Hole Spanners (SHS) connect otherwise isolated network communities, yet existing methods struggle with scalability and adaptability across diverse networks. This project introduces GraphSHS, a graph neural network model that reframes SHS discovery as a learning problem, leveraging network structure and node features for efficiency and accuracy. To enhance adaptability, Meta-GraphSHS employs meta-learning, enabling rapid adaptation to unseen graphs. Experiments on synthetic and real-world datasets show GraphSHS significantly outperforms existing methods in speed, while Meta-GraphSHS ensures high accuracy across diverse networks.

Microservice Security & Vulnerability Assessment

Microservice architectures enhance scalability and maintainability but introduce complex security risks, including expanded attack surfaces and intricate dependencies. This project systematically analyzes existing research on microservice vulnerabilities, identifying key risks and mitigation strategies. By synthesizing insights, it provides practical guidelines for securing microservices, advancing both research and industry best practices.

DDoS Defense in SDN Infrastructure

Software-Defined Networking (SDN) enhances network efficiency but remains highly vulnerable to DDoS attacks due to its centralized architecture. This project systematically reviews DDoS detection and mitigation strategies, analyzing attack types across SDN planes, security measures, and available datasets, tools, and simulators. By identifying open challenges and future research directions, it provides a comparative analysis of DDoS defense techniques, guiding advancements in SDN security.

Mechanism Design for Public Projects

This project explores mechanism design for both nonexcludable and excludable binary public projects, aiming to maximize consumer participation and agent welfare. For nonexcludable projects, it establishes conditions where the conservative equal costs mechanism is optimal and introduces a dynamic programming approach for optimal mechanism derivation. For excludable projects, it identifies cases where the serial cost-sharing mechanism is near-optimal for small groups and leverages machine learning-based techniques—integrating optimization, neural networks, and efficient training—to enhance mechanism performance, advancing the field of AI-driven mechanism design.

Smartphone Security from Cyber Threats

Smartphones, powered by GSM, GPRS, EDGE, and UMTS, are integral to daily life but face rising cyber threats due to their role in internet access, banking, and online transactions. This project analyzes wireless network vulnerabilities, exposing security risks associated with smartphone communication technologies. By identifying key attack vectors, it explores strategies to enhance mobile security frameworks, ensuring safer digital interactions for users.
