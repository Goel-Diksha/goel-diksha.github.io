Hardening Active Directory Graphs

The Defending Active Directory project represents a strategic advancement in redefining enterprise cybersecurity. In an era of evolving threats, Active Directory (AD), the backbone of identity and access control, remains a key target for attackers. This project approaches AD security as a strategic interaction between an intelligent defender and a dynamic adversary, leveraging AI techniques such as reinforcement learning and evolutionary optimization to anticipate and disrupt attack paths before they can be exploited. Shifting from reactive defense to proactive strategy, it offers an adaptive, intelligent framework to protect the core of modern digital infrastructure. Visit the project to explore more

Structural Hole Spanners: Key Connectors in Networks

This project focuses on discovering Structural Hole Spanners (SHSs), key nodes that bridge disconnected communities within a network. These nodes are crucial for enabling communication and influence flow, but existing methods often fail to scale or adapt across diverse network types. This project leverages graph-based machine learning to analyze both structural patterns and node features. It also incorporates adaptive strategies to generalize across unseen networks, aiming to deliver a fast, accurate, and scalable approach for identifying critical connectors in complex systems.

Microservice Security & Vulnerability Assessment

Microservice architectures offer exceptional scalability and flexibility, making them a cornerstone of modern software systems. However, this architectural shift introduces significant security challenges, including expanded attack surfaces and complex inter-service dependencies. This project conducts a comprehensive analysis of vulnerabilities within microservice environments, identifying key security risks and evaluating existing mitigation strategies. By integrating insights from both academic research and industry practice, it aims to provide practical, scalable guidelines for securing microservices, contributing to stronger, more resilient system design across diverse deployment contexts.

DDoS Defense in SDN Infrastructure

Software-Defined Networking (SDN) has become a transformative paradigm in modern network management, enabling greater flexibility, scalability, and control, especially in cloud computing, IoT, and large-scale systems. However, its centralized design also introduces significant security risks, particularly from Distributed Denial of Service (DDoS) attacks. This project explores the landscape of DDoS threats in SDN environments, examining attack vectors across various network planes and communication layers. It investigates existing detection and mitigation strategies, along with the tools and frameworks supporting current research, while identifying key challenges and opportunities for advancing SDN security.

Mechanism Design for Public Projects

At the intersection of economics, optimization, and AI, this project tackles one of the core challenges in public decision-making: designing fair and efficient mechanisms for public projects. It explores both nonexcludable and excludable binary public projects, with a focus on maximizing participation and promoting agent welfare. For nonexcludable projects, the project identifies optimal cost-sharing conditions and employs dynamic programming to derive effective mechanisms. For excludable projects, it explores near-optimal solutions and applies ML techniques to improve performance. These contributions advance AI-driven mechanism design by uniting theory with practical innovation.

Smartphone Security from Cyber Threats

As smartphones become the primary gateway to internet access, online transactions, and everyday communication, their exposure to cyber threats continues to grow. Powered by wireless technologies such as GSM, GPRS, EDGE, and UMTS, these devices support a wide range of services, from voice and messaging to banking and business applications. This project investigates the underlying vulnerabilities in smartphone communication technologies, uncovering the security risks that threaten user privacy and data integrity. By identifying critical attack vectors, it aims to inform the development of stronger mobile security frameworks, ensuring safer, more resilient digital experiences in an increasingly connected world.
